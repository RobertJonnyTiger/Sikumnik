כמה יחידות יקנה הצרכן אחרי שינוי מחיר?
P×Q₀ = הוצאה מוצא ידועה קשיח: P₁ > P₀ TE₁ > TE₀ → Q₁ = TE₁ / P₁ > TE₀/P₁ Q₁ > P₀×Q₀ / P₁
P×Q₀ = הוצאה מוצא ידועה יחידתי: P₁ > P₀ TE קבוע → Q₁ = TE₀ / P₁ Q₁ = TE₀ / P₁
P×Q₀ = הוצאה מוצא ידועה גמיש: P₁ > P₀ TE₁ < TE₀ → Q₁ = TE₁ / P₁ < TE₀/P₁ Q₁ < TE₀ / P₁
