# סיכומניק (Sikumnik) 📚

פרויקט ה"סיכומניק" הוא יוזמה לשילוב בינה מלאכותית (AI) בלמידה ושליפה של תובנות מתוך חומר אקדמי גולמי. המטרה היא לקחת מחברות, קבצי PDF, ומסמכים סרוקים, ולהפוך אותם לתוכן אינטראקטיבי, נגיש ומובנה היטב.

## 🎯 מטרות הפרויקט
- **הנגשת מידע:** הפיכת קבצים וחומר סרוק (כולל כתב יד) לטקסט דיגיטלי איכותי.
- **מבנה נתונים:** ארגון החומר בפרקים, דפי "Worked Examples" וטבלאות חישוב דינמיות.
- **אינטליגנציה מלאכותית:** שימוש במודלים מתקדמים (כגון Gemini) כדי "לראות" ולהבין מסמכים מורכבים כמו דוחות רווח והפסד בכתב יד.

## 🛠️ טכנולוגיות
- **Frontend:** נבנה באמצעות Next.js ו-React ליצירת חוויית משתמש מודרנית ועיצוב נקי.
- **Backend / Scripts:** שימוש ב-Python לעיבוד מקדים (Ingestion), OCR והפקת מבני נתונים (JSON) עבור המערכת.
- **AI Integration:** שימוש במודלים של ראייה ממוחשבת ושפה לפענוח התוכן.

## 📈 סטטוס נוכחי
הפרויקט מתמקד כרגע בתחומים:
1.  **חשבונאות:** המרת דוחות כספיים ופתרונות ידניים לפורמט דיגיטלי.
2.  **הסתברות:** תצוגה ויזואלית של עצי הסתברות ופתרונות.
