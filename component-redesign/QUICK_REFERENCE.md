## **IMPLEMENTATION CHECKLIST**

### Before You Code
- [ ] Pick your colors from the palette above
- [ ] Choose the right component for your teaching purpose
- [ ] Verify your color combinations for accessibility

### While You Code
- [ ] Use only brand colors (no custom colors)
- [ ] Use design tokens (CSS variables)
- [ ] Follow spacing scale
- [ ] Maintain border radius consistency

### After You Build
- [ ] Test contrast ratios (should be 4.5:1+)
- [ ] Test on light AND dark backgrounds
- [ ] Test on mobile (responsive)
- [ ] Get peer review on color choices
